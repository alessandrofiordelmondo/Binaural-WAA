if you want to change the HRTF set you have to replace the file inside the folder HRTF in the same path of the index.html.
Every HRTF must have the name as following:
NAME_A_P###_E_***
NAME : HRTF set name (only non-numerical character);
A : azimuth;
P : position L or R;
### : degree from 0 to 180 for each side;
E : elevation;
*** : elevation degree;
In this case we have only an example and we don't use the elevation, but only the azimuth localization
